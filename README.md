## 📌 Project Overview

This project, titled **"Intelligent Automated Medicine Information and Queue Management System with Prescription Scanner for Community Pharmacy,"** is a thesis/capstone project developed by 4th-year students of AISAT College Dasmariñas.

The system is designed to enhance pharmacy operations by integrating intelligent automation for queue management, prescription scanning, and accessible medicine information. It aims to improve service efficiency, reduce waiting time, and ensure accurate prescription processing within community pharmacies.


### ⚙️ Features
Queue Management System
Prescription Scanner (OCR)
Medicine Information Lookup
Admin Dashboard (if applicable)
Inventory Tracking (if included)

### 🛠️ Tech Stack
Frontend: React, Typescript, Tailwind CSS
Backend: Node JS, Express
Database: PostgreSQL
APIs/Services: Gemini API Services, Websocket, REST API


📊 System Modules
Queue Management
Prescription Processing
Medicine Database
User Management
